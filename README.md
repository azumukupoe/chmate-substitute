# ChMate Substitute

ChMateの書き込みを肩代わりするアプリ。  
荒らしに使われても面倒なのでこっそり使ってね。  
ChMate本スレ等にもあんまり晒さないでね:angry:

荒らし対策のためapkはビルドしません。  
Android Studio等でどうぞご勝手に。

ちなみにこのアプリはcgiにデータをPOSTしているだけで  
特に「5ch API」と言われるような認証が必要なAPIは使用していません。  
ですが問題があれば消します。

## 起動方法

- 「外部アプリで書き込む」オプション
- 書き込みボタン長押し
